# 易排 - simple-maker
- 一个自定义排名的工具，例如LOL英雄T级排行榜。  
- 高度自定义排序，拖拽排序，自定义等级，自定义图片等。

- A custom ranking tool, such as LOL hero T-level leaderboard。
-  Height custom sort, drag and drop sort, custom level, custom picture, etc.
